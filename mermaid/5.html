Homo Sovieticus vs Homo Zapiens

Центральная идея романа — трансформация "советского человека", ориентированного на идеологию, в "человека переключающего" (Homo Zapiens), чья личность формируется потоком медийных и рекламных образов.

classDiagram class Homo_Sovieticus { +Идеологическая вера +Ориентация на "светлое будущее" +Источник идентичности: Партия, коллектив +Главный страх: Быть врагом народа +Медиа: Пропаганда +Цель: Построение коммунизма } class Homo_Zapiens { +Цинизм и отсутствие веры +Ориентация на "здесь и сейчас" +Источник идентичности: Бренды, телеобразы +Главный страх: Выпасть из тренда, быть "лохом" +Медиа: Бесконечное переключение каналов +Цель: Потребление, получение "ВАУ-импульса" }

Механизм "ВАУ-импульса"

Эта цепочка описывает ключевой двигатель потребительского общества в романе. "ВАУ-импульс" — это иррациональное желание обладать товаром, вызванное рекламой, которое заменяет все другие стремления человека.
